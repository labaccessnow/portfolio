# Application Projects

Practical software that makes infrastructure and network operations easier to run.

## What I build
- **Python tooling & CLIs** — provisioning, monitoring, and reporting utilities for the network and
  virtualization estate.
- **Vendor-API integrations** — pulling and correlating data from network and platform REST APIs
  for custom reporting.
- **Retrieval over operational knowledge** — a local **semantic-search / RAG** service over my own
  operational notes and documentation (embeddings + vector search), so answers come from real
  context instead of guesswork.

## Where the field is (2024–2026)
**RAG (retrieval-augmented generation)** moved from novelty to standard practice — embeddings +
vector search (FAISS, vector databases) grounding LLM answers in your own data. In operations
specifically, **LLM-assisted AIOps** is now real: incident triage, root-cause analysis, and
log-anomaly detection built on retrieval over runbooks and telemetry. My tooling sits at that
intersection — small, dependable automation that increasingly uses retrieval/LLM techniques to make
operational knowledge searchable and actionable.

Engineering-adjacent software — written to solve real operational problems, not demos.
