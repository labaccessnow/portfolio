# Cisco ACI

Application Centric Infrastructure — spine-leaf data-center fabric design, policy,
and automation.

## Focus
- **Logical design** — tenants, VRFs, bridge domains, EPGs, and contracts that express
  application intent rather than VLAN plumbing.
- **Fabric automation** — programmatic configuration through the APIC REST API.
- **Service integration** — connecting compute and L4-L7 services into the fabric.
- **Migration** — moving workloads from traditional three-tier to a spine-leaf fabric.

Part of broader data-center and network-security engineering.
